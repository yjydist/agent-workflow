# Completion Gates

## Design Package

- Goal, users, non-goals, constraints, and risks are captured.
- Adapter classification is recorded.
- Domain model, workflows, architecture, interfaces, and data notes are coherent.
- Quality attributes and validation strategy are explicit.

## Current Target

- Included work is small enough to implement independently.
- Excluded and deferred work are written down.
- Acceptance boundaries are testable.
- Freeze status is clear.

## Handoff Ready

- Implementation plan is sequenced and scoped.
- Review notes have no blocking issues.
- Handoff entry point lists required reading order.
- Stop conditions tell the receiving agent when to ask for clarification.
