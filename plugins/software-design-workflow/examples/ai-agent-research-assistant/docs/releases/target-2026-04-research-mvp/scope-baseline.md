# Scope Baseline

## Target

`target-2026-04-research-mvp`

## Included

- Accept one research topic and an optional audience hint.
- Gather notes from approved sources only.
- Produce one short markdown brief with citations and unresolved questions.
- Record enough trace data to review source usage during validation.

## Excluded

- Autonomous scheduling.
- Shared workspaces.
- Paid or credentialed source integrations.
- Long-term memory beyond local notes for the current brief.
