# Browser runtime adapter

## Layer

Runtime

## Applies when

Use when code runs in browser contexts, including SPAs, extensions, service workers, or web workers.

## Design focus

Define browser support, routing mode, storage, network assumptions, security boundaries, performance budgets, and accessibility constraints.

## Expected artifacts

Browser matrix, client storage model, network and caching behavior, performance and a11y constraints.

## Legacy migration

Legacy `frontend-spa` runtime content migrates here when it concerns browser execution.
