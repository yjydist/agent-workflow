# Local Machine runtime adapter

## Layer

Runtime

## Applies when

Use when code runs on a user or developer workstation with local files, shells, OS permissions, or installed dependencies.

## Design focus

Define install model, filesystem access, environment detection, config paths, update flow, and OS compatibility.

## Expected artifacts

Install and update notes, filesystem boundaries, OS compatibility, local config model.

## Legacy migration

Legacy `cli-tool` local execution content migrates here.
