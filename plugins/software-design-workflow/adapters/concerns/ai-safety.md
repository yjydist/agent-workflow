# AI Safety concern adapter

## Layer

Concern

## Applies when

Use when LLMs, embeddings, agents, generated content, autonomy, or model decisions affect users or systems.

## Design focus

Define model boundaries, prompt risks, tool-use approval, harmful output controls, evaluation, monitoring, human override, and data exposure limits.

## Expected artifacts

Safety boundaries, evaluation plan, tool approval matrix, monitoring and incident response.

Legacy `ai-agent` safety-boundaries and evaluation content migrates here.
