# Legacy compatibility assets

This directory contains retained legacy compatibility assets for the old `rag-app` project-type bundle.

For new work, prefer adapter guidance instead of treating this directory as a primary entry point.

Adapter mapping: `rag-app` -> `knowledge-system + http-api`, `web-ui`, or `agent-tools`, with AI and retrieval concerns added explicitly.

Use these files only as transitional reusable material during migration.
