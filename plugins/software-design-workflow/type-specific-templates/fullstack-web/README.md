# Legacy compatibility assets

This directory contains retained legacy compatibility assets for the old `fullstack-web` project-type bundle.

For new work, prefer adapter guidance instead of treating this directory as a primary entry point.

Adapter mapping: `fullstack-web` -> `application + web-ui + http-api + server`.

Use these files only as transitional reusable material during migration.
