# Legacy type-specific templates

This directory contains retained legacy compatibility assets from the older single project-type model.

New workflows should prefer adapter guidance in `../adapters/` and treat those adapter docs as the primary source for current design work.

The files under each legacy project-type directory are kept only as transitional reusable materials during migration. Reuse them when they still help, but do not treat this tree as the preferred entry point or the primary modeling system.
