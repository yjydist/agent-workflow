# Skill: Test Design

## Goal

定义项目如何测试.

## Test Types

- Unit tests
- Integration tests
- API tests
- UI tests
- CLI tests
- SDK examples
- AI evaluation cases
- Manual acceptance tests

## Output Template

```md
# Test Plan

## Test Scope

## Test Matrix

## Required Automated Tests

## Manual Tests

## Out of Scope
```

## Rules

- 测试计划必须对应 V1 范围.
- 每个关键功能至少有一条正向和一条失败测试.
