# Skill: Error Handling Design

## Goal

统一定义错误类型,错误响应和恢复策略.

## Checklist

- 用户输入错误
- 权限错误
- 数据不存在
- 数据冲突
- 配置错误
- 文件系统错误
- 网络错误
- 第三方服务失败
- LLM / embedding 调用失败
- 超时和重试
