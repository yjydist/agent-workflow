# Type-specific Documents

This directory contains project-type-specific specifications.

Examples:

- `backend-api/`: API, database, auth, deployment
- `frontend-spa/`: pages, components, routing, state management
- `cli-tool/`: commands, flags, config, input/output
- `ai-agent/`: model usage, prompts, tools, memory, evaluation
- `rag-app/`: ingestion, chunking, embedding, retrieval, generation, evaluation

Only create the subdirectories that match the current project type.
