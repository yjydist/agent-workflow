# Skill: Acceptance Criteria Design

## Goal

定义做到什么程度算完成.

## Acceptance Template

```md
## Feature / Phase

### Setup

### Steps

### Expected Result

### Failure Cases
```

## Rules

- 验收标准必须具体.
- 尽量给出命令,URL,输入样例,输出样例.
- 不写"体验良好"这种无法判定的标准.
- 每个 Phase 都要有验收方法.
