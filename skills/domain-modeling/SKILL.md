# Skill: Domain Modeling

## Goal

识别项目中的核心概念,关系和状态.

## Output Template

```md
# Domain Model

## Entities

### EntityName

- Meaning:
- Key attributes:
- Lifecycle:

## Relationships

## State Machines

## Assumptions

## Open Questions
```

## Checklist

- 每个核心名词是否都有定义?
- 实体之间的关系是否清楚?
- 状态是否枚举完整?
- 状态转移是否有规则?
- 是否区分了领域模型和存储模型?
