## Must Clarify

- 页面列表
- 用户路径
- 组件层级
- 状态来源:local state / global store / server state
- 表单校验
- loading / empty / error states
- API contract

## Anti-overdesign

不要一开始设计复杂 design system,除非项目目标就是 UI 组件库.
